# 2021_bigdatacampus_competition
## 2021년도 서울시 빅데이터 캠퍼스 공모전 참가
### 서울시 불법 주정차 문제 해결을 위한 최적의 주차장 입지 선정

- 심각한 불법 주정차 문제 인식
- 환경변수(거주인구, 유동인구, 주차장수)를 분석 후 가중치를 설정하여 행정동 선정
- 입지선정 알고리즘 활용하여 주차장 입지 선정
